How it works

1

Run the scanner

Point MCPShield at your MCP config file. Works with Claude Desktop, Cursor, VS Code, Windsurf, and any MCP client.

npx mcpshield scan --config claude_desktop_config.json
2

Review findings

Get a detailed report of every vulnerability, typosquat, credential leak, and misconfiguration ‚Äî with severity ratings and remediation advice.

3

Add to CI/CD

Use the GitHub Actions workflow to scan MCP configs on every push and PR. Block merges on critical findings.

npx mcpshield scan --json --output report.json
4

Ship with confidence

Exit code 0 means your config is clean. Exit code 1/2 means issues need attention. Integrate into any pipeline.
